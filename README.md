# Exchange Rate Prediction and Macroeconomic Analysis

## Overview
This project develops predictive models to analyze exchange rate movements between the U.S. Dollar and major U.S. trading partners using macroeconomic indicators and financial market variables.

## Objective
The goal of this project is to:
- Identify key drivers influencing exchange rate fluctuations
- Analyze relationships between macroeconomic indicators and currency movements
- Develop predictive models for exchange rate forecasting
- Generate actionable insights for investors, businesses, and policymakers

## Countries Analyzed
- Australia
- Canada
- Eurozone
- Japan
- Great Britain
- United States

## Macroeconomic Variables Used
- Exchange Rates
- CPI (Inflation)
- Real GDP
- Unemployment Rates
- Crude Oil Prices
- Gold Prices
- S&P 500
- VIX (Volatility Index)

## Tools & Technologies
- R
- Linear Regression
- Panel Linear Models (PLM)
- Predictive Analytics
- Financial Analytics
- Time Series Analysis
- Econometrics
- Data Visualization
- R Markdown

## Data Sources
- FRED (Federal Reserve Economic Data)
- Macroeconomic datasets from 2013–2023

## Data Preprocessing
- Missing value imputation
- Frequency conversion (monthly/quarterly to daily)
- Percentage change calculations
- Relative difference calculations
- Outlier handling
- Lag variable creation

## Models Used

### Linear Regression
Used to capture relationships between exchange rates and macroeconomic indicators.

### Panel Linear Models (PLM)
Used to analyze time-series and cross-sectional country-level relationships simultaneously.

## Key Findings
- S&P 500 and VIX were among the strongest predictors of exchange rate movements.
- Inflation and unemployment rates significantly influenced currency fluctuations.
- Lagged macroeconomic indicators improved model performance.
- Exchange rate volatility increased significantly during the COVID-19 period.

## Business Impact
This project demonstrates practical applications of predictive analytics in:
- Currency risk management
- International trade planning
- Investment decision-making
- Hedging strategy development
- Economic policy analysis
